## [1.0.0] - 2025-11-25

### Добавлено
- Базовая функциональность Messari API клиента для NestJS
- NestJS модуль `MessariModule` с поддержкой `forRootAsync`
- Сервис `MessariService` для работы с Messari API
- Поддержка всех основных методов Messari API:
  - `getTopAssets()` - получение списка топ криптовалютных активов
  - `getAsset()` - получение детальной информации об активе включая метрики и профиль
  - `getAssetNews()` - получение новостей связанных с конкретным активом
  - `getGeneralNews()` - получение общих новостей криптовалютного рынка
  - `searchAssets()` - выполнение поиска активов по запросу
- Поддержка API ключа для аутентификации (обязательно)
- Конфигурируемые таймауты и базовый URL
- Полная типизация TypeScript с экспортируемыми типами
- 100% покрытие тестами
- Обработка ошибок с детальным логированием
- Интеграция с `@makebelieve21213-packages/logger`
